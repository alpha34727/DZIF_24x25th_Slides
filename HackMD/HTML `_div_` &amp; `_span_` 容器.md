# HTML `<div>` &amp; `<span>` 容器

## `<div>` 區塊元素容器

`<div>`沒有特別的意義，只作為容器(區塊元素)使用，用來包住其他HTML標籤，方便與 CSS 或 JavaScript 作互動

### 語法：

```htmlembedded!
<div>區塊內容</div>	
```

## `<span>` 行內容器

`<span>` 和 `<div>` 類似，不具有特別的意義，通常用於文字或其他行內(inline)的標籤，方便與 CSS 或 JavaScript 作互動

### 語法：

```htmlembedded!
<span>文字</span>	
```

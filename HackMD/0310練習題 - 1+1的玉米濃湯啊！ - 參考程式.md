# 0310練習題 - 1+1的玉米濃湯啊！ - 參考程式

```python=
money = int(input())
raw = money
con = 0; fri = 0; ham = 0

while(True):
    if money >= 40:
        money -= 40
        con += 1
    else: break

    if money >= 35:
        money -= 35
        fri += 1
    else: break

    if money >= 75:
        money -= 75
        ham += 1
    else: break

print(f'在總共花了{raw-money}元後，陳小廷買了{con}杯玉米濃湯，{fri}包薯條，{ham}個漢堡。')
```

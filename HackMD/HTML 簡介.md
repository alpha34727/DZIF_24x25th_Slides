# HTML 簡介

**HTML**（全稱：**H**yper**T**ext **M**arkup **L**anguage，中文：超文本標記語言），與Python、C++等我們一般學的程式設計語言不同，HTML是標記語言。

HTML常與CSS、JavaScript一起用於設計網頁，HTML掌管了網頁的架構

## 歷史

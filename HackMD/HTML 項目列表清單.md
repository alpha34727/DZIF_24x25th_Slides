# HTML 項目列表清單

---

# `<li>` 清單項目

在`<ul>`和`<ol>`內部排列，不限數量

---

# `<ul>` 無序清單
(unordered lists)

---

### 範例：

```htmlembedded=
<ul>
    <li>星期一</li>
    <li>星期二</li>
    <li>星期三</li>
</ul>
```

---


<ul>
    <li>星期一</li>
    <li>星期二</li>
    <li>星期三</li>
</ul>

---

## `<ol>` 有序清單
(ordered lists)

---

### 範例：

```htmlembedded=
<ol>
    <li>步驟一</li>
    <li>步驟二</li>
    <li>步驟三</li>
</ol>
```

---

<ol>
    <li>步驟一</li>
    <li>步驟二</li>
    <li>步驟三</li>
</ol>

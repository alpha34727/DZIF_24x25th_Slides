# CSS 起手式

### 在資料夾中新增一個CSS檔案

![](https://i.imgur.com/D31Vb2r.gif =100%x)

:::info
註：副檔名要是「`.css`」
:::

### 在`<head></head>`中新增以下語句

```htmlembedded!
<link rel="stylesheet" href="到CSS檔案的路徑">
```

範例：
![](https://i.imgur.com/jGP5i2C.gif =100%x)

### 如此一來，就可以在新增的CSS檔案中編寫CSS了
